 Library Book Management System

A simple Library Book Management System developed in C++ to manage library records efficiently. This project provides an 
easy-to-use, menu-driven interface for performing basic library operations. It is designed for beginners to practice
Object-Oriented Programming (OOP) concepts.

FEATURES:

- Add Book
- View All Books
- Search Book
- Issue Book
- Return Book

Technologies Used:

- C++
- Object-Oriented Programming (OOP)

How to Run:

1. Download or clone the project.
2. Open it in any C++ IDE (Code::Blocks, Dev-C++, Visual Studio, or VS Code).
3. Compile and run the program.
4. Use the menu to manage library books.


CONCEPTS USED:

- Classes and Objects
- Functions
- Conditional Statements
- Loops

PURPOSE:

This project was created to demonstrate the basic concepts of C++ by building a simple system for managing library books. It helps users perform common library tasks while improving programming and problem-solving skills.
